What is the differnce between Margin,Border and padding

The information written above is what surrounds the space between the image or text also known as the content.

  • Margin is the space outside of the border and padding. It can overlap other elements.
  • Border is the surrounding space outside of the padding and content. It's what gives the element a border.
  • Padding is the space between the border and the content, it allows an open area in between the properties.


The text below is how you may apply the properties in your CSS file. The div is the element.


div{
margin: 20px;
border: 2px solid #color;
padding: 10px;
}

What is best practice when using Classes and ids

  • Id is for an individual element while class can be used for multiple elements under the same class.
  • When it comes to naming the elements you can use the same name for an ID and class because they have their own significance.
  • It's best to not use Id when linking to CSS it is more suitable for a Javascript file because Id can only be used once per page.

What is the difference between Relative, absolute and Fix positioning

  • Relative positioning allows you to adjust the element moving up, down, left, or right.
  • Absolute positioning is an element that is positioned to its nearest positioning ancestor.
  • Fixed positioning will stay fixed to the page even when you scroll on the page.

Display inline vs inline block

  • block element takes up more space than a inline elements, the border continuse after the text. But the parent limits the size to the rest of the blocks.

The text below is how you may use it in CSS


div{
display:block;
}

  • Diplay inline will display next to one another or side to side. It only takes up as much room necerssary, wraps around the text.

The text below is how you may use it in CSS


div{
display:inline;
