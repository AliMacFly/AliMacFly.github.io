JavaScript is a way of add complex feature to a web page or website. Javascript adds functionality to enhance the user experince.


Flow and loops, arrays, and objects

  • Flow and loops - Flow control is the order in which statement are executed in a program. Loop is a specific type of flow, loops repeatedly executes a block of code just as long as if it's specific condicition remains true.

    • Flow example: 'if', 'else', and 'elif'

    Loop example: 'for', and 'while'

  • Arrays - Arrays allows to create a list of elements, with each element having a unique index. Arrays stores its elements in square brackets and can contain zero or more elements. Arrays are fundamentally used for storing and manipulating collection a of data.
  • Objects - Objects useful storing information, it a way of organizing data and putting all the related data into a function.

    Functions and why they are useful

    Function are reusable blocks of code, that executes a specific task, it is design to be reuse instead of rewriting the block code. Functions are useful for organizing code, which creates structured and efficient programing.


    DOM and DevTools

    DOM stands for Document Object Model, which is an programmign interface for a web document. The DOM provides a way of manipulating the content and structure of an HTML or XML document as a tree like object.
    Developer Tools also known as DevTools, it is a set of debugging and development tools that is built into modern web browsers. DevTools have a range of functions including modifying and adjust HTML, CSS and JavaScript files.
